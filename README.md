# xim-pkgindex-moonbitlang

xim package index for MoonBit community

## 添加索引仓库

> 添加索引仓库到xim的索引仓库管理器中

```bash
xim --add-indexrepo moonbitlang:https://github.com/frozenlemontee/xim-pkgindex-moonbitlang.git
xim --update index
```

## 使用仓库中的包

```bash
# 自动下载安装MoonBit命令行工具
xlings install moonbitlang:moonbit-cli
```
